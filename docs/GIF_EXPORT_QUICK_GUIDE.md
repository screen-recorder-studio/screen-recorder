# GIF 导出快速指南

## 🚀 快速开始

### 1. 点击 "Export GIF" 按钮
在视频录制完成后，点击紫色的 "Export GIF" 按钮。

### 2. 配置导出参数
对话框会弹出，提供以下选项：

#### 🎯 快速预设（推荐）
- **💎 高质量**: 最佳画质，文件较大
- **⚖️ 平衡**: 默认选项，质量和大小平衡
- **💾 小文件**: 最小文件，适合网页分享
- **🌊 流畅**: 高帧率，动画流畅
- **🕹️ 复古**: 像素风格，怀旧感

#### ⚙️ 自定义参数
- **帧率 (FPS)**: 5-30，默认 10
- **采样质量**: 1-30，默认 10（越小越好）
- **缩放比例**: 25%-100%，默认 75%
- **工作线程**: 1-8，默认 2

### 3. 开始导出
点击 "开始导出" 按钮，对话框会显示实时进度。

### 4. 等待完成
导出完成后，对话框自动关闭，GIF 文件自动下载。

## 📊 进度阶段

导出过程分为 5 个阶段：

1. **准备中** (0-10%): 初始化编码器
2. **合成视频** (10-80%): 合成背景和视频
3. **提取帧** (10-80%): 从视频提取帧
4. **添加帧** (80-95%): 添加帧到 GIF
5. **渲染 GIF** (95-100%): 最终渲染

## ⚠️ 注意事项

1. **导出时无法取消**: 导出开始后，取消按钮会被禁用
2. **对话框保持打开**: 导出完成前，对话框不会关闭
3. **自动下载**: 导出完成后，文件会自动下载

## 🎨 进度显示

对话框中会显示：
- ✅ 当前阶段（中文）
- ✅ 进度百分比
- ✅ 已处理帧数 / 总帧数
- ✅ 动画进度条
- ✅ 旋转加载图标

## 🔧 故障排除

### 文件太大？
- 降低帧率（10 → 8）
- 减小尺寸（75% → 50%）
- 降低质量（10 → 20）

### 导出太慢？
- 增加线程（2 → 4）
- 降低质量（10 → 15）
- 减小尺寸（75% → 50%）

### 颜色有色带？
- 启用抖动（Floyd-Steinberg）

## 🎯 最佳实践

### 1. 选择合适的预设
- 不确定？使用 **⚖️ 平衡**
- 需要分享？使用 **💾 小文件**
- 追求质量？使用 **💎 高质量**

### 2. 根据需求调整
- 文件大小优先 → 降低帧率和缩放
- 质量优先 → 提高质量和帧率
- 速度优先 → 增加线程数

### 3. 查看预估信息
- 导出前检查预估文件大小
- 确认输出尺寸是否合适
- 确认帧数是否正确

## ❓ 常见问题

**Q: 为什么对话框不关闭？**
A: 导出过程中对话框会保持打开，显示进度。导出完成后会自动关闭。

**Q: 可以取消导出吗？**
A: 目前导出开始后无法取消，请在点击"开始导出"前确认参数。

**Q: 进度卡住了怎么办？**
A: 请耐心等待，某些阶段（如渲染 GIF）可能需要较长时间。

**Q: 导出失败了怎么办？**
A: 检查浏览器控制台的错误信息，可能是内存不足或浏览器限制。
