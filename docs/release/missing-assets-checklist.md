# 缺失资产清单

以下是发布到 Chrome Web Store 前还需要准备的视觉资产和材料。

## 🖼️ 必需的视觉资产

### 1. 屏幕截图 (必需)
**规格要求**:
- 数量：1-5 张
- 尺寸：1280x800 像素 或 640x400 像素
- 格式：PNG 或 JPEG
- 用途：在 Chrome Web Store 中展示扩展功能

**建议内容**:
- [ ] 主界面截图（侧边栏面板）
- [ ] 录制选择界面（标签页/窗口/屏幕选择）
- [ ] 元素选择功能演示
- [ ] 录制进行中的界面
- [ ] 视频导出/保存界面

### 2. 宣传图块 (推荐)
**规格要求**:
- 尺寸：440x280 像素
- 格式：PNG 或 JPEG
- 用途：Chrome Web Store 推广展示

**设计建议**:
- 突出 "Screen Recorder Studio" 品牌
- 展示核心功能图标
- 使用吸引人的视觉设计
- 包含 "for SAAS" 定位信息

### 3. 小型宣传图块 (可选)
**规格要求**:
- 尺寸：440x280 像素
- 格式：PNG 或 JPEG
- 用途：小尺寸推广展示
