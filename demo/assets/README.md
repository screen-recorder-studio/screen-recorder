# Assets Directory

This directory contains the extension icons and other assets.

## Required Icons

- icon16.png - 16x16 pixel icon for the extension
- icon48.png - 48x48 pixel icon for the extension  
- icon128.png - 128x128 pixel icon for the extension

## Note

For development purposes, you can create simple placeholder icons or use any 16x16, 48x48, and 128x128 pixel PNG images.

The extension will work without these icons, but Chrome may show warnings in the developer tools.
