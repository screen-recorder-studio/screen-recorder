# 需求文档 - MVP版本

## 介绍

SaaS视频录制器Chrome扩展的MVP版本，专注于核心功能：屏幕录制和基础背景替换。目标是快速实现可用的产品演示视频录制工具，支持简单的背景替换和视频居中显示功能。

## 需求

### 需求 1 - 基础录制功能

**用户故事：** 作为SaaS产品的演示者，我希望能够通过Chrome扩展录制屏幕内容，以便创建产品演示视频。

#### 验收标准

1. 当用户点击扩展图标时，系统应显示简单的录制控制面板
2. 当用户点击开始录制时，系统应录制当前标签页内容
3. 当用户点击停止录制时，系统应结束录制并自动下载WebM格式视频文件
4. 系统应显示录制状态（录制中/已停止）

### 需求 2 - 基础背景替换

**用户故事：** 作为内容创作者，我希望能够为录制的视频添加简单的背景，以便提升视频的视觉效果。

#### 验收标准

1. 当录制完成后，系统应提供背景选择选项
2. 系统应提供3-5个预设的纯色背景选项
3. 当用户选择背景时，系统应将原始视频居中显示在背景上
4. 系统应在视频四周添加固定的内边距（如20px）
5. 当背景应用完成时，系统应自动下载处理后的视频

### 需求 3 - 简单用户界面

**用户故事：** 作为Chrome扩展用户，我希望有一个简洁的界面，以便快速完成录制和背景替换。

#### 验收标准

1. 扩展弹窗应包含录制按钮和状态显示
2. 录制完成后应显示背景选择界面
3. 系统应显示处理进度
4. 界面应适配扩展弹窗的标准尺寸（约400x300px）
