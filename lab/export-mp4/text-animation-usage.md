# 📝 文字动画 MP4 导出使用指南

## 🚀 快速开始

### 1. 打开页面
在浏览器中打开 `text-animation-demo.html` 文件。

### 2. 基本操作流程
1. **编辑文字内容** - 在"主要文字"输入框中输入您想要的文字
2. **选择动画类型** - 从6种动画效果中选择一种
3. **调整文字样式** - 设置字体大小、字体族、颜色等
4. **配置动画参数** - 调整动画速度和行间距
5. **设置导出参数** - 选择视频质量、帧率和时长
6. **开始动画预览** - 点击"开始动画"按钮查看效果
7. **导出 MP4** - 点击"导出文字动画 MP4"按钮

## 🎨 功能详解

### 文字内容设置
- **多行文字支持**: 使用换行符分隔多行文字
- **实时预览**: 显示文字行数、字符数和内容预览
- **中英文混合**: 支持中文、英文和特殊字符

### 6种动画效果

#### 1. 打字机效果 (Typewriter)
- 逐字符显示文字
- 带有闪烁光标
- 适合展示代码或重要信息

#### 2. 淡入效果 (Fade In)
- 逐行淡入显示
- 每行有延迟效果
- 适合优雅的文字展示

#### 3. 滑入效果 (Slide In)
- 文字从右侧滑入
- 带有缓动效果
- 适合动态展示

#### 4. 缩放效果 (Scale Up)
- 文字从小到大缩放
- 逐行显示
- 适合强调重点内容

#### 5. 彩虹文字 (Rainbow)
- 每个字符颜色动态变化
- 彩虹色彩循环
- 适合创意和艺术效果

#### 6. 弹跳文字 (Bounce)
- 每个字符上下弹跳
- 错位动画效果
- 适合活泼有趣的内容

### 文字样式控制
- **字体大小**: 20px - 120px 可调
- **字体族**: 8种常用字体可选
- **文字颜色**: 支持任意颜色选择
- **背景颜色**: 支持任意背景色
- **行间距**: 1.0 - 3.0 倍可调

### 动画参数
- **动画速度**: 0.5x - 3.0x 可调
- **实时调整**: 修改参数后立即生效

### 导出设置
- **视频质量**: 
  - 标清 (1 Mbps) - 适合网络分享
  - 高清 (2.5 Mbps) - 推荐设置
  - 超清 (5 Mbps) - 高质量需求
- **帧率**:
  - 24 FPS - 电影标准
  - 30 FPS - 网络视频标准（推荐）
  - 60 FPS - 高流畅度
- **录制时长**: 3-30 秒可调

## 🎬 导出流程

### 准备阶段
1. 确保文字内容和样式设置满意
2. 选择合适的动画类型
3. 调整导出参数

### 导出过程
1. 点击"导出文字动画 MP4"按钮
2. 系统会显示进度条和状态信息
3. 导出过程中请勿关闭页面
4. 等待"导出完成"提示

### 预览和下载
1. 导出完成后会显示视频预览
2. 可以直接在页面中播放预览
3. 点击"下载 MP4 文件"保存到本地
4. 文件名格式: `text-animation-[动画类型]-[时间戳].mp4`

## 💡 使用技巧

### 文字内容建议
- **行数控制**: 建议不超过6行，避免文字过小
- **字符数量**: 每行建议不超过20个字符
- **内容规划**: 重要信息放在前面几行

### 样式搭配建议
- **高对比度**: 使用深色背景配浅色文字，或相反
- **字体选择**: 
  - Arial/Helvetica - 现代简洁
  - Times New Roman - 正式文档
  - Impact - 强调效果
- **颜色搭配**: 避免背景色和文字色过于接近

### 动画选择指南
- **商务演示**: 推荐淡入效果或滑入效果
- **创意展示**: 推荐彩虹文字或弹跳文字
- **技术演示**: 推荐打字机效果
- **标题展示**: 推荐缩放效果

### 导出参数建议
- **社交媒体**: 30 FPS, 高清质量, 5-10秒时长
- **演示文稿**: 24 FPS, 高清质量, 3-8秒时长
- **网站展示**: 30 FPS, 标清质量, 3-5秒时长

## 🔧 故障排除

### 常见问题

#### 1. 动画不显示
- 检查浏览器是否支持 Canvas
- 确保 JavaScript 已启用
- 刷新页面重试

#### 2. 导出失败
- 检查浏览器兼容性（推荐 Chrome/Edge）
- 确保有足够的内存空间
- 尝试降低视频质量或缩短时长

#### 3. 文字显示不完整
- 调整字体大小
- 检查 Canvas 尺寸是否足够
- 减少文字行数或字符数

#### 4. 颜色显示异常
- 检查颜色值格式是否正确
- 尝试使用标准颜色值
- 确保背景色和文字色有足够对比度

### 性能优化
- **减少文字数量**: 过多文字会影响性能
- **合理设置时长**: 过长的视频会消耗更多资源
- **选择合适质量**: 根据需求选择合适的视频质量

## 📚 技术说明

### 支持的浏览器
- ✅ Chrome 90+
- ✅ Edge 90+
- ⚠️ Firefox (部分功能)
- ❌ Safari (不支持)

### 技术特性
- 基于 Canvas 2D API 渲染
- 使用 WebCodecs API 进行视频编码
- MediaBunny 库提供 MP4 封装
- 支持硬件加速编码

### 文件格式
- 输出格式: MP4 (H.264 编码)
- 分辨率: 800x600 像素
- 颜色空间: RGB
- 容器: MP4 标准格式
